## Jekyll 블로그란?
Jekyll은 정적 사이트 생성기로, 마크다운 파일을 HTML로 변환해주는 도구입니다.

### 장점

- **빠른 로딩**: 정적 파일이므로 매우 빠름
- **GitHub Pages 지원**: 무료로 호스팅 가능
- **마크다운 지원**: 간편한 글 작성
- **테마 다양성**: 다양한 테마 사용 가능

### 설치 방법

```bash
# Ruby 설치 (macOS)
brew install ruby

# Jekyll 설치
gem install jekyll bundler

# 새 블로그 생성
jekyll new my-blog
cd my-blog

# 로컬 서버 실행
bundle exec jekyll serve
```
